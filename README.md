# 六爻起卦工具

> 以传统古法金钱卦摇卦、起卦的六爻工具。本工具仅呈现卦象与卦辞，供《易经》学习者参研易理，不提供任何吉凶解读。

---

## 功能说明

- **六步摇卦**：模拟三枚铜钱摇卦计数，每摇一次得一爻，从初爻到上爻依次生成
- **动爻标识**：爻位变动自动标记为动爻
- **三卦联动**：自动推算并显示本卦、互卦、变卦
- **卦辞爻辞**：内置《周易》64卦卦辞及爻辞，随卦象完整呈现
- **一键复制**：生成完整卦象文本，方便保存记录
- **键盘快捷键**：支持快捷键操作

---

## 使用步骤

1. 打开页面，在「占问之事」框中输入事由（选填）
2. 点击「摇卦」按钮，或按键盘 **空格键**，逐次摇出六爻（共六次）
3. 六爻完成后，自动显示本卦、互卦、变卦及对应卦辞、爻辞
4. 点击「复制」按钮，或按键盘 **C键**，将卦象结果复制保存
5. 点击「重置」按钮，或按键盘 **R键**，确认后重新起卦

## 占问须知

《易经》为圣贤穷理尽性之书，起卦需诚敬。念起于心，则卦随心感。古有以下原则，当知非所宜则不占：

### 五不占

| 原则 | 含义 |
| :--- | :--- |
| **不诚不占** | 心不虔诚、态度轻慢，感应不通，占亦无验 |
| **不义不占** | 违背道义、损人利己之事，不必问卦，问亦不告 |
| **不疑不占** | 已有明确判断且无疑虑，无需起卦；《易》为决疑而设 |
| **不正不占** | 动机不纯、心念不正，卦象虽现，其意难明 |
| **亵渎不占** | 同一事反复追问，敬心已失，问则无应 |

> **注**：卦象所示，皆为启发反省之机，非宿命定论。

>
> 卦辞爻辞均引自通行本《周易》，请以学术和文化传承的视角使用本工具，理性看待卦象，以卦为镜，反观自身，明则通达。以“易”之象，明己之心，修己之德，君子乾乾，夕若惕。
